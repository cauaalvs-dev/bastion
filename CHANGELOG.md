## [1.0.0] — 2025-01-01

### Added
- Initial project structure with Next.js 14 + TypeScript + Tailwind CSS
- Authentication: register, login, logout with bcrypt
- JWT access tokens + refresh token rotation
- RBAC middleware (admin, user, viewer roles)
- Dashboard with user profile CRUD
- Supabase integration with Row Level Security
- Security headers via next.config.js (CSP, HSTS, X-Frame-Options)
- SECURITY-REPORT.md with 8 documented vulnerabilities
- GitHub Actions CI/CD pipeline
- AWS infrastructure documentation
