# Security Policy

## Supported versions

| Version | Supported |
|---|---|
| main | ✅ |

## Reporting a vulnerability

This is a personal portfolio project for security research purposes.

If you find a real vulnerability in the live deployment, please report it via email before public disclosure. I aim to respond within 48 hours.

Do not open a public GitHub issue for security vulnerabilities.

## Security research scope

This project intentionally includes historical vulnerabilities that have been documented and fixed. See [SECURITY-REPORT.md](./SECURITY-REPORT.md) for the full log.

The live deployment runs the fixed version only.
